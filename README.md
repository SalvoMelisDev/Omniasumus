Omniasumus
Piattaforma Full-Stack integrata per la gestione di contenuti spirituali, ricettari e mini-games in TypeScript. Il progetto implementa un'architettura moderna con separazione netta tra Backend (Spring Boot) e Frontend (Angular), garantendo sicurezza e scalabilità.

Stack Tecnologico
Backend
Java & Spring Boot: Core dell'applicazione e gestione della logica di business.

Spring Security: Gestione avanzata dell'autenticazione e delle autorizzazioni.

RBAC (Role-Based Access Control): Sistema di permessi granulari per gradi (User/Admin) per le operazioni CRUD su testi e ricette.

Maven: Gestione delle dipendenze e build automation.

Frontend
Angular: Framework per la creazione di un'interfaccia utente dinamica e reattiva.

TypeScript: Linguaggio principale per lo sviluppo dei mini-giochi e della logica client-side.

CSS/SCSS: Styling personalizzato per un'esperienza utente fluida.

Funzionalità Principali
Sezione Spirituale: Visualizzazione e gestione di testi dedicati, con permessi di modifica basati sul grado utente.

Ricettario Dinamico: Modulo per inserire, modificare e visualizzare ricette (perfetto per chi ama cucinare, magari con l'air fryer!).

Gaming Hub: Mini-giochi sviluppati in TypeScript con sistema di salvataggio dei record per utente.

Gestione Utenti: Dashboard per la gestione dei profili e dei punteggi ottenuti nei giochi.
